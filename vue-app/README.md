# Vue 3 + TypeScript + Vite

### 起動手順(install/dev/build/preview)
1.  依存関係のインストール
``` bash
npm run dev
```

2. 開発サーバー起動
```bash
npm run dev
```

3. ビルドとプレビュー
- ビルド
```bash
npm run build
```

- プレビュー
```bash
npm run build
```

### main.tsとApp.vueの役割
## main.tsの役割
アプリケーションの起動をするための入口（エントリーポイント）

Vueアプリの作成して、どのコンポーネントを画面に表示するかを設定する。

システムの設定
ライブラリの接続
動かすための準備をするところ

## App.vueの役割
Vueアプリケーションの一番上のコンポーネント
UIの全体構造
共通レイアウト
見せるための土台を作るところ
